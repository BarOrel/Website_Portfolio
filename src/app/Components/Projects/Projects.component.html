Social Media (At Work)

a social media website that allows users to create posts, like and comment on posts, and follow other users. It has a modern user interface and a database to store and retrieve user data. The app is built using ASP.NET on the backend, which uses SQL Server for data storage, and Angular on the frontend

Chat Application

A fully-functional chat application, built using Angular for the client-side and an ASP.NET API and SQL database for the back-end, featuring multiple advanced capabilities.

E Commerce Computer Store

An e-commerce website was built using ASP.NET and Angular. The homepage displays products, categories, and promotions, and includes search, cart, and checkout functions for customers. An admin panel was created for product and order management.
